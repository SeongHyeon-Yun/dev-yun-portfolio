# 포트폴리오

## 프로젝트 소개

### 1. 랜딩 페이지 제작

- 서비스 홍보를 위한 랜딩 페이지 구현
- 사용자 유입 및 문의 전환을 고려한 구조 설계

### 2. B2B 도매몰 및 주문 관리 시스템

- 상품 개별 주문 및 대량 주문 기능 구현
- 관리자 페이지를 통한 상품 및 주문 관리 프로세스 구축
- 입/출금 관련 트랜잭션 처리 및 데이터 정합성 확보
- 대량 주문 처리 시 큐 기반 로직을 적용하여 성능 개선

### 3. 스포츠 경기 일정 서비스

- 외부 API를 활용한 경기 일정 및 정보 제공 기능 구현
- 대용량 API 호출 구조 최적화 및 병목 구간 개선
- 데이터 처리 성능 개선 (O(n) → O(log n))

### 4. 스포츠 경기 승리 확률 예측 시스템

- 외부 API를 통해 상대 전적, 배당, 경기 결과 데이터를 수집하여 승리 확률 예측 시스템 구축
- 배당 데이터를 24시간 단위로 수집 및 DB 저장
  1. 대용량 데이터 저장 과정에서 SQL 최적화 및 병렬 처리(스레드) 적용

- 배당 기반 확률 계산 로직 구현
  1. 배당의 역수(1 / odds)를 활용하여 암묵적 확률 계산
  2. 승/무/패 확률의 합이 1보다 큰 구조(북메이커 마진)를 확인
  3. 전체 합으로 나누어 정규화하여 실제 확률로 변환

  예시)
  승 = 1 / 2.00 = 0.5  
   무 = 1 / 3.00 = 0.333  
   패 = 1 / 4.00 = 0.25

  정규화:
  p = (1 / odds) / Σ(1 / odds)

- 정규화된 확률을 기반으로 가장 높은 값을 승리 예측 결과로 활용
